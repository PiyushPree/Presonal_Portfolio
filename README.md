It showcases my skills, projects, and experience as a Java Backend Developer & Full-Stack Engineer.

✨ Features

👨‍💻 About Me – Short intro & career background

📂 Projects Section – Highlights of my top projects with live demos

🛠️ Skills Section – Technical skills across Backend, Frontend, and Databases

📜 Resume Download – One-click CV download

📞 Contact Form – Directly reach me through email

📱 Responsive Design – Optimized for mobile, tablet, and desktop
